「Maru-Jan 麻雀脳力塾3 牌効率編 無料版」について(SignalTalk Inc.)

■牌効率とは

Maru-Janのお贈りする牌効率は、あがりまでの最も効率の良い打ち筋のことを指しています。
「Maru-Jan 麻雀脳力塾3 牌効率編 無料版」では、全ての捨牌に対しての有効牌と残り枚数を表示することにより、視覚的に牌効率を表せる様になりました。

■動作環境、推奨環境

OS対応状況		：Windows 2000/XP/Vista
CPU			：Intel PentiumⅡ 266MHz 以上(PentiumⅢ 500MHz以上推奨)
メモリ			：128MB以上(256MB以上推奨)

※グラフィックカード、サウンドカード等のドライバプログラムは、
　最新のバージョンをご使用になることを推奨します。

※本製品の動作環境の他に、
　ご使用のOSの動作環境を満たしている必要があります。

※現在、Macintosh対応版はありません。

■起動方法

インストーラによるインストール作業の終了後、デスクトップに配置される、
「Maru-Jan 麻雀脳力塾3 牌効率編 無料版」という名前のショートカットアイコンをダブルクリックして起動します。
アイコンが表示されない場合は、
スタートメニューから、
「プログラム」→「Maru-Jan 麻雀脳力塾3 牌効率編 無料版」→
「Maru-Jan 麻雀脳力塾3 牌効率編 無料版で遊ぶ」を選んで下さい。

※「Maru-Jan 麻雀脳力塾3 牌効率編 無料版」以外のソフトを全て終了して、起動して下さい。

■遊び方

1. 「練習開始」、「試験開始」のどちらかを選ぶとゲームが開始します。
　 どちらのモードを選択されても牌効率をお楽しみ頂けます。

2. カーソルキー(「←」・「→」)やマウスで捨てたい牌を選択し、
　「Enter」キー、もしくは左クリックで選択された牌を捨てます。

3. 27巡以内に和了をしなければ流局となります。
　 ※ただし、槓をした場合は、1回毎に自模牌が1つ減ります。

4. 麻雀脳力試験モードでは、10局を打ち終えると最後に成績が表示されます。

5. 初段までは飛び級で段位が取得できますが、二段より上の段位を取得するためには、順番に段位を取得する必要があります。
　 (例：三段を取得するためには、二段を取得する必要があります。)
　 何度も挑戦し、最高成績を目指しましょう！

6. 成績閲覧にて、総合成績や最高成績が閲覧できます。

■アンインストールの方法

スタートメニューから、
「設定」→「コントロール パネル」→「プログラムの追加と削除」→
「プログラムの変更と削除」を選んで「Maru-Jan 麻雀脳力塾3 牌効率編 無料版」を選択し、
削除して下さい。
