
「Maru-Jan 麻雀点数塾 無料版」 について
　　　　　　　　　　　　　　　(SignalTalk)

■動作環境

○Windows

　OS対応状況　　　　　　：Windows 98.2000.XP.Vista

　CPU 　　　　　　　　　：Intel PentiumⅡ 266MHz 以上(PentiumⅢ 500MHz以上推奨)

　メモリ　　　　　　　　：128MB以上(256MB以上推奨)

　ハードディスク　　　　：空き容量30MB以上

　モニタ　　　　　　　　：1024×768ピクセル以上

　グラフィックカード　　：VRAM 2MB以上 (3D機能は必要ありません。)

　入力機器　　　　　　　：キーボード、マウス対応

※グラフィックカード、サウンドカード等のドライバプログラムは、
　最新のバージョンをご使用になることを推奨します。

※本製品の動作環境の他に、
　ご使用のOSの動作環境を満たしている必要があります。

※「四人打ちネットワーク対戦はこちら」バナーをクリックする際は、インターネットに接続できる環境が必要になります。

○Macintosh

現在、対応版はありません。


■起動方法

インストーラによるインストール作業の終了後、デスクトップに配置される、
「Maru-Jan 麻雀点数塾 無料版」という名前のショートカットアイコンをダブルクリックして起動します。
アイコンが表示されない場合は、
スタートメニューから、
「プログラム」→「Maru-Jan 麻雀点数塾 無料版」→「Maru-Jan 麻雀点数塾 無料版で遊ぶ」
を選んで下さい。

※「Maru-Jan 麻雀点数塾 無料版」以外のソフトを全て終了して、起動して下さい。


■遊び方

1. メニューの試験開始を選び、「開始する」ボタンを押すと試験が始まります。

2. マウスで「数字」ボタンをクリックして点数を入力します。(キーボードから数字を直接入力することもできます)
　「C」ボタン、または「Delete」「Backspace」キーを押すことで一桁戻すことができます。

3. 「決定」ボタン、または「Enter」キーを押すと答え合わせになります。

4. 15問終えると、成績が表示されます。
 　※正解時間には不正解時の経過時間は含まれません

5. メニューの成績閲覧にて、段位取得回数や総合成績が閲覧できます。

6. 初段までは飛び級で段位が取得できますが、二段より上の段位を取得するためには、順番に段位を取得する必要があります。
　 (例：三段を取得するためには、二段を取得する必要があります。)
　 計算力・判断力を鍛えて、免許皆伝を目指しましょう！


■アンインストールの方法

スタートメニューから、
「設定」→「コントロール パネル」→「プログラムの追加と削除」→「プログラムの変更と削除」を選んで
「Maru-Jan 麻雀点数塾 無料版」を選択し、削除して下さい。
